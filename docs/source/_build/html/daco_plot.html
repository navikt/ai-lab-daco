plotDistributionsOfVariable(variable, filename_prefix='')[source]¶

Plotting distributions for numerical and categorical variables in dataframes. Plots are saved in file_dir. The plots include error bars and visualization of deviation from the real dataset.

Parameters:
  • variable (str) – name of variable plotting histogram for
  • filename_prefix (str) – prefix in filename
